1) Project Title

*Blinkit Sales & Outlet Performance Dashboard (Power BI)*

2) Short Description / Purpose

This project presents an interactive Power BI dashboard built to analyze *Blinkit’s sales performance, outlet distribution, and product insights*. The goal is to help stakeholders quickly understand key business metrics, identify trends, and make data-driven decisions.

3) Tech Stack

* *Power BI* – Data visualization & dashboard creation
* *DAX (Data Analysis Expressions)* – Calculated measures & KPIs
* *Power Query (M Language)* – Data cleaning & transformation
* *Excel / CSV Dataset* – Data source

4) Data Source

* Sample retail dataset inspired by Blinkit operations
* Includes:

  * Sales data
  * Item categories
  * Outlet types, sizes & locations
  * Ratings & item visibility

5) Features / Highlights

*  KPI Cards for:

  * Total Sales ($472K)
  * Average Sales ($141)
  * Number of Items (3350)
  * Average Rating (3.9)

*  Interactive Visuals:

  * Outlet Establishment trend over years
  * Sales breakdown by Item Type
  * Fat Content distribution
  * Outlet Size & Location analysis
  * Outlet Type comparison table

*  Dynamic Filters:

  * Outlet Location Type
  * Outlet Size
  * Item Type

*  Drill-down insights using slicers and visual interactions

6) Business Impact

* Helps identify *top-performing product categories*
* Enables comparison of *outlet performance by type & location*
* Supports *inventory and sales optimization decisions*
* Improves visibility into *customer preferences (ratings & fat content trends)*

7) Key Insights

* Tier 3 outlets contribute the *highest overall sales ($472K)*
* Supermarket Type 2 shows *highest sales among outlet types*
* Fruits, Snacks, and Household items are *top revenue generators*
* Low-fat vs regular products show *balanced contribution*
* Average ratings are consistent (~3.9), indicating *stable customer satisfaction*
